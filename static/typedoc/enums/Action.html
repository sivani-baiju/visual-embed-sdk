The list of actions that can be performed on visual ThoughtSpot entities, such as answers and Liveboards.

This enum is used to specify the actions that could be disabled, hidden or made visible.

example
const embed = new LiveboardEmbed('#embed-container', {
   ... // other options
   visibleActions: [Action.Save, Action.Explore],
   disableActions: [Action.Save],
   hiddenActions: [Action.Download], // Set either this or visibleActions
})
