Optional getAuthToken

  • getAuthToken(): Promise<string>
    • Defined in types.ts:240

    [AuthServer] A function that invokes the trusted authentication endpoint and returns a Promise that resolves to the auth token string. For trusted authentication, the authEndpoint or getAuthToken attribute is required.

    It is advisable to fetch a new token inside this method and not reuse and older issued token, as because when auth expires this is called again and if it is called with an older token the authentication will not succeed.

    Returns Promise<string>
