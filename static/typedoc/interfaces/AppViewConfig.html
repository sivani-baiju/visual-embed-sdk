Optional pageId

The application page to set as the start page in the embedded view.

Optional path

path: string
  • Defined in embed/app.ts:104

A URL path within the app that is to be embedded. If both path and pageId attributes are defined, the path definition takes precedence. This is the path post the #/ in the URL of the standalone ThoughtSpot app. Use this to open the embedded view to a specific path.

For eg, if you want the component to open to a specific liveboard you could set the path to pinboard/<liveboardId>/tab/<tabId>.

example
<AppEmbed path="pinboard/1234/tab/7464" />
